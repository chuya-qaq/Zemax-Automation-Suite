# Zemax 智能设计辅助专家 (Zemax AI Assistant) - 开发说明文档

这是一个基于 **Python 3.8+** 和 **Zemax ZOSAPI** 开发的工程化光学辅助设计套件。本项目旨在通过模块化的工具箱，帮助光学工程师自动化处理繁琐的数据提取、系统诊断及复杂的降敏优化任务。

> **⚠️ 开发状态说明：**
> 本项目目前处于**半成品/原型验证阶段**。核心逻辑已打通，但部分高级功能（如 AI 演化算法）仍需根据具体光路结构进行参数微调。代码结构已完成解耦，旨在为开发者提供一个可扩展的二次开发框架。

---

## 🚀 已实现的核心功能

### 1. 🖥️ 工程化 GUI 交互平台

* **前后端解耦：** 使用 `ui_core`（Tkinter）与 `zemax_core`（ZOSAPI 逻辑）分离的架构，相比传统的单脚本工具，具有更好的稳定性。
* **多页面管理：** 内置“智能诊断”与“自动降敏”两个核心功能模块，支持实时日志输出与进度监控。
* **线程安全：** 耗时任务（如 Monte Carlo 模拟或本地优化）在后台线程运行，确保 GUI 界面不卡死。

### 2. 🤖 智能标签诊断 (Smart Labeler)

* **数据驱动分析：** 自动解析 Zemax 公差报告中的 `Worst Offenders`。
* **专家规则库：** 结合 Seidel 系数与光线追迹数据（AOI、光线高度），自动为光学表面标注功能标签：
* `Dominant_Source`: 识别像差的主导源头。
* `High_AOI_Hazard`: 预警入射角过大导致的高敏感区域。
* `Potential_Sponge`: 发现可吸收余量像差的“潜力海绵”面。


* **处方建议：** 针对不同标签提供具体的优化修改方向。

### 3. 🧬 绝热演化降敏 (Adiabatic Evolution)

* **非对称打击策略：** 针对识别出的“像差源头”施加不平衡压力，强制系统跳出当前的病态平衡。
* **自适应动力学：** 内置动态步长（$\lambda$）控制，根据 MTF 的波动自动触发“贪婪加速”或“回滚呼吸”机制。
* **物理边界保护：** 在演化过程中通过操作数监控 CTGT/ETGT/OPLT，防止透镜相交或光线跑飞。

### 4. 🛠️ 自动化工具集成

* **一键优化准备：** 快速将玻璃设为 `Substitute` 模式，或将所有非零非球面系数设为变量。
* **高精度报告：** 一键导出包含 LDE 数据、Seidel 系数分布以及全视场 30 lp/mm 平均 MTF 的综合报告。

## 📂 项目结构

```text
my project/
├── zemax AI assistant/       # 主程序目录
│   ├── main.py               # 程序入口
│   ├── ui_core/              # UI 逻辑层
│   │   ├── app.py            # 主框架与页面调度
│   │   └── page_analysis.py  # 诊断页面逻辑
│   └── zemax_core/           # ZOSAPI 核心层
│       ├── controller.py     # 核心控制器，管理连接与分发

```

## 📝 开发者备注（半成品局限性）

* **连接路径：** `zemax_core/config.py` 中的 `ZEMAX_PATH` 需根据本地安装路径手动配置。
* **算法鲁棒性：** 目前的绝热演化算法主要针对透射式摄影镜头优化，对于大角度离轴或反射系统可能需要调整评估函数模板。
* **UI 细节：** 部分高级配置项（如多阶段演化参数）尚未完全开放至 GUI，需在 `services_ai.py` 中手动修改。
