# Zemax 智能设计辅助专家 (Zemax AI Assistant)

这是一个高度模块化、工程化的 Zemax 辅助设计套件。通过将 GUI 界面与 ZOSAPI 核心逻辑深度解耦，本项目提供了一个稳定、可扩展的光学设计增强环境。

## 🏗️ 模块化架构

项目采用层次化设计，确保逻辑清晰且易于维护：

### 1. 🖥️ UI 核心层 (`ui_core`)
- **`app.py`**: 主框架管理，负责页面切换与全局生命周期。
- **`page_analysis.py`**: 专注于像差、MTF 及系统数据的可视化分析页面。
- **`page_desens.py`**: 降敏与绝热演化控制中心，管理复杂的演化参数。
- **`theme.py`**: 统一的界面视觉风格定义。

### 2. ⚙️ Zemax 核心逻辑层 (`zemax_core`)
- **`services_ai.py`**: 核心智能识别引擎，集成了更为复杂的专家诊断逻辑与标签识别。
- **`services_ana.py`**: 高精度的光学分析服务，涵盖 LDE、Seidel、MTF 等数据提取。
- **`services_tol.py`**: 专业的公差文件解析器，支持多种编码格式。
- **`controller.py`**: 核心控制器，负责 ZOSAPI 的生命周期管理及各服务间的任务调度。

## 🌟 核心特性

- **专家级系统诊断：** 结合公差分析结果，深度解析系统敏感度分布，精准定位“像差源头”与“吸收海绵”。
- **先进演化算法：** 内置改进版的“绝热演化”控制逻辑，支持多阶段打击目标设定，自动处理优化过程中的呼吸式恢复。
- **工程化交互设计：** 相比单脚本工具，提供了更好的异常处理、线程安全及响应式 UI 体验。
- **一键自动化流：** 从玻璃库替换到变量锁定，覆盖了光学设计中大量重复性手工操作。

## 🛠️ 安装与运行

1. **环境准备：** 
   - Python 3.8+ 
   - 已安装 Ansys Zemax OpticStudio (建议 2024 R1 及以上版本)
2. **依赖安装：**
   ```bash
   pip install pythonnet pandas
   ```
3. **启动程序：**
   运行项目根目录下的 `main.py`：
   ```bash
   python main.py
   ```

## 📝 开发者备注
本项目支持二次开发。如需添加新的分析模块，可在 `zemax_core` 下新建 Service 并在 `ui_core` 中注册相应的 Page。
